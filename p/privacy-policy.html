Kebijakan Privasi

Kebijakan Privasi ini menjelaskan bagaimana aplikasi SiCatat Hutang mengelola dan melindungi data pengguna. Dengan menggunakan aplikasi ini, Anda dianggap telah membaca dan menyetujui kebijakan ini.

1. Informasi yang Dikumpulkan

Aplikasi SiCatat Hutang tidak mengumpulkan, menyimpan, atau mengirimkan data pribadi pengguna ke server mana pun. Semua data yang Anda masukkan, seperti nama, nominal, tanggal, dan status hutang atau piutang, disimpan secara lokal di perangkat Anda.

2. Penggunaan Data

Data yang Anda masukkan digunakan hanya untuk menampilkan dan mengelola catatan hutang dan piutang di dalam aplikasi. Data tersebut tidak digunakan untuk tujuan lain dan tidak dibagikan kepada pihak ketiga.

3. Penyimpanan Data

Semua data disimpan secara lokal di perangkat Anda menggunakan penyimpanan internal aplikasi. Aplikasi ini tidak menggunakan layanan cloud, akun pengguna, atau sinkronisasi online.

4. Keamanan Data

Kami berusaha menjaga data Anda tetap aman dengan menggunakan mekanisme penyimpanan standar pada sistem Android. Namun, karena data disimpan di perangkat pengguna, keamanan juga bergantung pada keamanan perangkat itu sendiri.

5. Penghapusan Data

Anda dapat menghapus data kapan saja melalui fitur yang tersedia di dalam aplikasi, seperti menghapus catatan atau menggunakan fitur reset. Setelah dihapus, data tidak dapat dipulihkan.

6. Perubahan Kebijakan Privasi

Kebijakan Privasi ini dapat diperbarui dari waktu ke waktu. Setiap perubahan akan ditampilkan di halaman ini. Kami menyarankan Anda untuk meninjau halaman ini secara berkala.
