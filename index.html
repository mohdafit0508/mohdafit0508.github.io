SiCatat Hutang

Catat dan kelola hutang piutang secara sederhana dan offline.

Tentang Aplikasi

SiCatat Hutang adalah aplikasi sederhana untuk membantu Anda mencatat dan mengelola hutang serta piutang secara rapi. Aplikasi ini dirancang untuk penggunaan pribadi maupun usaha kecil yang ingin memantau catatan keuangan secara praktis tanpa koneksi internet.

Dengan SiCatat Hutang, Anda dapat menyimpan data hutang dan piutang langsung di perangkat Anda, sehingga catatan tetap tersedia kapan saja dan tidak bergantung pada server atau akun online.

Fitur Utama

  • Mencatat hutang dan piutang secara terpisah
  • Menyimpan nama, nominal, tanggal, dan status pembayaran
  • Menandai catatan sebagai lunas atau belum lunas
  • Mengedit dan menghapus catatan
  • Menampilkan daftar berdasarkan tanggal dan status
  • Mode tema terang dan gelap
  • Data tersimpan secara lokal di perangkat

Penggunaan

Aplikasi ini cocok digunakan untuk kebutuhan pribadi, keluarga, atau usaha kecil yang ingin mencatat transaksi hutang dan piutang secara lebih teratur.

SiCatat Hutang tidak memerlukan login, tidak menggunakan akun, dan tidak mengirim data ke server. Semua data disimpan dan dikelola langsung di perangkat Anda.
